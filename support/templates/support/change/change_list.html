17.03.2022
  • Раздел курса теперь можно скрыть от студентов до назначения задания по задаче из этого раздела.
    Скрытые разделы будут отображаться для студентов в списке разделов курса под названием «Скрытый раздел» без возможности перейти на страницу просмотра раздела. Скрытый раздел станет доступен для просмотра тем студентам, которым преподаватель назначит задания по задачам из этого раздела.
  • Результаты автоматической проверки посылок к заданиям с ограниченным временем приёма посылок теперь не отображаются студентам до истечения этого времени.
    Например: если указать время приёма посылок в задании (или заданиях при случайном распределении) по задаче типа «варианты ответа» , то студенты увидят результаты автоматической проверки и оценку только после истечения указанного времени.
05.03.2022
  • Курсы, разделы и задачи теперь при удалении попадают в корзину.
    Находящиеся в корзине курсы, разделы и задачи можно восстановить или удалить безвозвратно. Доступ к корзине (по ссылке в выпадающем меню в заголовке главной страницы) имеют только модераторы.
  • Разделы и задачи теперь можно перемещать в другие курсы и разделы, к которым у преподавателя есть доступ.
    Чтобы переместить раздел (или задачу) в другой курс (или раздел) выберите пункт выпадающего меню "Переместить" в заголовке страницы перемещаемого курса (или задачи).
  • Добавлен факультет "МФК".
    Курсы в этом факультете допускают назначение ведущих преподавателей из других факультетов, а таблицы заданий содержат дополнительный фильтр по факультету. Начинать и завершать курсы на факультете "МФК" можно отдельно для каждого потока другого факультета, так и для всех потоков сразу.
  • Добавлен тип задачи "Устный ответ".
  • Разрешена загрузка файлов с расширениями .djvu и .sav в разделы и задачи.
  • Увеличено с 5 до 20 максимальное количество заданий для каждого студента, генерируемое при случайном распределении заданий.
  • Комментарии к заданию теперь отображаются на странице задания.
  • Теперь студенты могут отправлять посылки только к задачам по назначенным им заданиям.
    После завершения курса, все задания по задачам из этого курса считаются завершенными и не допускают отправки новых посылок.
